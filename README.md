# Frontend Assignment

[Demo](http://frontend-assignment.surge.sh)

#### Projeyi Çalıştırma

``` bash
# repo'yu bilgisayarınıza çekin
$ git clone https://github.com/suleymandtr/frontend-assignment.git

# npm bağımlılıklarını yükleyin
$ npm install

# projeyi development ortamında başlatın
$ npm run serve
```
